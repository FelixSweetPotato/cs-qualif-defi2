## Prerequisites
install poetry
```bash
curl -sSL https://install.python-poetry.org | python3 -
```
or 
```bash
pip install poetry
```

## Project Setup

Poetry will automatically create a virtual environment and install all dependencies:
```bash
poetry install
```

### Run the Application

```bash
poetry run python main.py
```
